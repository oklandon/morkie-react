# morkie-react
The beloved `<marquee>` reimagined in React

# Installation
`npm install morkie-react`

# Usage
```js
import Morkie from 'morkie-react';

export default class SomeComponent extends React.Component {
  render(){
    return(
      <Morkie text={`some sort of text`}/>
    );
  }
}
...
```
